# 湖癔科技自动生成系统

这是一个**纯前端多场景文档自动生成工具集**，支持内部通知、商务函件、维权通知等场景的一键生成，无需后端依赖，全设备自适应，加载速度快。


## 仓库文件结构
仓库包含7个核心HTML文件，对应系统的不同功能页面：
| 文件名                | 功能说明                     |
|-----------------------|------------------------------|
| `index.html`          | 系统首页（工具入口导航页）   |
| `code_20260112.html`  | 内部通知生成器               |
| `code_20260113.html`  | 商务警告函生成器             |
| `code_20260114.html`  | 举报维权通知生成器           |
| `code_20260115.html`  | 社交群管通知生成器           |
| `about.html`          | 关于我们页面                 |
| `contact.html`        | 联系我们页面                 |


## 使用方法
1. **访问方式**：
   开启GitHub Pages后，通过以下地址访问系统：
   `https://你的GitHub用户名.github.io/仓库名/`（替换“你的GitHub用户名”和“仓库名”为实际信息）

2. **工具功能**：
   - 内部通知生成器：支持员工管理、系统通知、安全预警等15+企业内部场景
   - 商务警告函生成器：生成规范的商务侵权、违约警告函件
   - 举报维权通知生成器：适配平台举报、权益维护的通知模板
   - 社交群管通知生成器：用于社群管理的公告、规则通知


## 系统特点
- 秒级加载：纯前端静态资源，无后端请求，访问响应快
- 多端适配：支持手机、平板、电脑等设备的自适应显示
- 全自定义：生成的内容支持在线编辑、复制、导出
- 无依赖：仅依赖基础HTML/CSS/JS，无需安装额外环境
